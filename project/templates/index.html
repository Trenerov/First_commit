Мои навыки программирования

Основано на пройденных темах курса

1. Основы программирования

Умею устанавливать и настраивать IDE, работать с консольным вводом и выводом данных, создавать простые объекты и работать с числами.

2. Условные конструкции

Использую условные операторы if, else для управления потоком выполнения программы.

3. Циклы

Уверенно применяю циклы for, while для повторения действий.

4. Типы данных и массивы

Понимаю основные типы данных, работаю со списками и многомерными массивами.

  • Целые числа
  • Строки
  • Булевы значения
  • Массивы (одномерные и двумерные)

5. Функции и процедуры

Могу писать собственные функции, передавать параметры, возвращать значения.

6. Обработка исключений и работа с API

Освоил базовую обработку ошибок через конструкции try-except и получил представление о работе с внешними API.

7. Основы ООП

Изучил основы объектно-ориентированного программирования: создание классов, объектов, конструкторов и методов.

8. Ввод и вывод данных

Уверенно читаю и записываю данные из/в консоль, а также обрабатываю пользовательский ввод.

9. Работа с изображениями

Ознакомился с базовыми принципами работы с изображениями в коде.
